# Privacy Policy for Onliner

**Last Updated:** December 16, 2025

Onliner is is a simple system utility which does not handle any data.

**1. Data Collection**
We do not collect, store, or transmit any personal data or usage statistics to external servers. The application functions entirely offline. The app has outgoing connection entitlements only to handle the internet connection availability check when the user enable it.

**2. Third-Party Services**
Onliner does not integrate with any third-party analytics or tracking SDKs.
